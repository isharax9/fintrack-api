# Task 04: Dashboard And Quick Add

Goal: build the first production dashboard and the main transaction creation workflow.

## Dashboard Scope

- Total account balance.
- Monthly income.
- Monthly expenses.
- Net cash flow and cash-flow margin.
- Spending breakdown.
- Recent transactions.
- Budget progress.
- Upcoming recurring bills.
- Savings goal highlight.
- Useful empty states for new users.

## Quick Add Scope

- Income/expense segmented control.
- Amount entry.
- Title/merchant.
- Account.
- Category.
- Tags.
- Date.
- Notes.
- Save and create another.

## Acceptance Criteria

- Dashboard uses real API data only.
- Quick add creates transactions and updates account balances.
- Loading, empty, error, and retry states exist.
- Mobile layout remains usable.

## Dependencies

- Task 01 API maturity.
- Task 02 design system.
- Task 03 app shell.
