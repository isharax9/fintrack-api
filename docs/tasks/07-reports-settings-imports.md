# Task 07: Reports Settings And Imports

Goal: complete advanced product workflows around insight, configuration, and data movement.

## Reports Scope

- Monthly summary.
- Category breakdown.
- Income vs expense trend.
- Budget variance.
- Account balance trend.
- Cashflow forecast using recurring transactions.
- PDF and CSV export.

## Settings Scope

- Profile.
- Currency.
- Password/security.
- Sessions.
- Notifications.
- Data export.
- Delete account.

## Imports Scope

- CSV upload.
- Column mapping.
- Preview.
- Duplicate detection.
- Category/account assignment.
- Import history.

## Acceptance Criteria

- Reports match backend totals.
- Settings reflect real API capabilities.
- CSV import can be safely previewed before commit.
- Error states are clear and recoverable.

## Dependencies

- Task 01 OpenAPI, audit logs, auth/session maturity.
- Task 05 accounts and transactions.
