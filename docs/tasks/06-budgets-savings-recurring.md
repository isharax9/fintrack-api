# Task 06: Budgets Savings And Recurring

Goal: finish the planning layer of FinTrack.

## Budgets Scope

- Monthly budget planner.
- Category budgets.
- Spent, remaining, percent used.
- Over-budget states.
- Grid and list mode.
- Rollover history when API supports it.

## Savings Scope

- Savings bucket.
- Savings goals.
- Allocate funds.
- Funding workflow.
- Goal progress and deadlines.

## Recurring Scope

- List recurring transactions.
- Create/edit/pause/delete recurring templates.
- Frequency and nextDate.
- Upcoming bills.
- Run history when API supports it.

## Acceptance Criteria

- Recurring page uses `/api/recurring`.
- Budgets and savings use real API data.
- Empty states guide users through setup.
- Mobile views remain usable.

## Dependencies

- Task 01 recurring API and savings contract.
- Task 02 design system.
